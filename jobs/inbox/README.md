# inbox/ — drop job postings here

Put one job posting per file (URL on the first line, then the posting text),
then run `/job-process`. The `/job-intake` command can create these files for
you.

This folder is empty in the sample because the demo already processed its two
postings — you can see the results in `../ready/`, `../blocked/`, the tracker,
and `../processed/`. The original two postings are archived in `../processed/`
if you'd like to copy one back here and replay the demo.

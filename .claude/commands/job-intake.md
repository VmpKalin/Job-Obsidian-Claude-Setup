Help me add a new job posting to the pipeline.

1. Ask me for the posting URL and the posting text (or read what I paste).
2. Work out a short filename from the company and role, like
   `company-role.md` (lowercase, dashes).
3. Create that file inside `jobs/inbox/` with the **URL on the first line**,
   followed by the full posting text.
4. Confirm the filename you created. Do not process it yet — that's
   `/job-process`.

Remember: only write inside `jobs/` (see `CLAUDE.md`).
